DEPARTMENTS

POST /api/departments

Register New Department

Request

           
          {
            "department_name": String,
            "department_abbreviation": String,
            "no_of_programs": Number,
            "lat": Number,
            "lng": Number
          }
          Authorization: Bearer {token}

Response

          
            {
              "statusCode": Number,
              "message": String
            }

PROGRAMS

Register New Program

Request

           
          {
            "program_title": String,
            "type": String,
            "starting": Date,
            "ending": Date,
